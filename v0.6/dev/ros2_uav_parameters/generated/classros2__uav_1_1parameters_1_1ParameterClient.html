class ParameterClient : public rclcpp::Node

A class for managing client-side ROS2 parameters.

This class handles the registration and synchronization of parameters with a remote parameter server.

Public Functions

ParameterClient(const std::string &node_name, const std::vector<std::string> &required_parameters, const std::string &server_name = "")

Construct a new Parameter Client object.

Parameters:

  • node_name – The name of the client node.

  • required_parameters – A list of required parameters to be synchronized.

  • server_name – The name of the parameter server node (default is “”). An empty string indicates the default parameter server. (i.e., node_namespace/parameter_server)
