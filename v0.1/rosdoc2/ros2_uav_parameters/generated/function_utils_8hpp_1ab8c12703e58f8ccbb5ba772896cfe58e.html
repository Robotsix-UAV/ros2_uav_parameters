std::string uav_ros2::utils::doubleToStringTrimZero(double value)

Converts a double to a string and trims trailing zeros.

This function takes a double value, converts it to a string, and removes any trailing zeros from the fractional part.

Parameters:

value – The double value to be converted to string.

Returns:

A string representation of the double value without trailing zeros.
